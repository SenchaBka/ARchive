# ARchive

**ARchive** is a location-based AR experience where people can leave digital “memories” (audio, images, and 3D content) pinned to real places, while others can discover them by physically walking to those locations.

## What ARchive Does

ARchive turns real-world locations into interactive story points.

Instead of saving memories in a private gallery or a generic feed, ARchive attaches them to places that actually matter, like a street corner, a campus building, a park bench, or any spot with meaning. When someone visits that location, they can unlock and experience what was left there.

---

## Core Features

- **Location-based discovery**
  - Content is tied to real GPS locations and becomes meaningful when you visit the place in person.

- **Create and drop memories**
  - Leave audio and media at a specific location for others to find.

- **AR-style experience**
  - Designed around exploration and “unlocking” content while moving through the world.

- **Web-first access**
  - Runs directly in the browser so people can try it quickly without installing an app.

---

## Tech Stack

This project is built as a modern web app.

- **Frontend:** Next.js / React
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Core concept:** Location-based AR content discovery
